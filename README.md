Anime Search App adalah aplikasi mini untuk mencari anime menggunakan **Jikan API**.  
Aplikasi ini dibuat dengan **React 19**, **TypeScript**, **Redux**, dan **TailwindCSS**, sesuai spesifikasi YoPrint.

---

## Fitur Utama

- **Halaman Pencarian**
  - Instant search dengan debounce 400ms
  - Cancel request sebelumnya menggunakan AbortController
  - Skeleton loader saat loading
  - Grid responsive dan flex-wrap
  - Pagination lokal dengan Prev/Next button
  - Empty state friendly
  - Error handling saat API gagal

- **Halaman Detail Anime**
  - Menampilkan informasi detail anime (judul, gambar, rating, genre, synopsis)
  - Bisa diakses dari SearchPage
